Construction Permit Visualizer



View project: constructionwhere.info


This web app plots the thousand most recent building permits approved in the city of Seattle. We use an AJAX call to the city of Seattle open data API then use the coordinates from the received JSON to plot permits on a map provided by the Google Map API.


My contributions for this project included building the server, page routing, filters for the permits, AJAX call, JSON manipulation, and styling for the map.
